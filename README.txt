Cimmerian, Marshini Mansion

Marcelle Brooks
March 2024

Collaborative group project designed to be an inclusive and accessible video game
prototype for the visually impaired. It is designed to use the keyboard as a 
physical space in which the user can move around to find their way out of a 
haunted mansion.

Headphones recommended. 
Collaborators: 
    - Thomas Chang
    - Cayde Bruce
    - Deanna Kelly

Necessary Libraries:
